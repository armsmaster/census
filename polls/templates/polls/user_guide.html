Инструкция

Как создать опрос

1. Заходим в справочник опросов (пункт "Опросы" в меню наверху)

2. Нажимаем на кнопку "Создать новый опрос"

3. Вводим название опроса и текст приветствия. Нажимаем сохранить.

Опрос создан, но он пустой (т.е. в нем нет ни одного вопроса)

Как заполнить опрос вопросами

1. Заходим в справочник опросов (пункт "Опросы" в меню наверху)

2. В списке опросов выбираем нужный. Идем по ссылке "Параметры опроса".

3. На странице опроса нажимаем на кнопку "Добавить вопрос". Появляется желтая форма.

4. В выпадающем списке выбираем тот вопрос, который хотим добавить. Нажимаем кнопку "Добавить"

ВНИМАНИЕ: добавить можно только тот вопрос, который уже существует в базе (в справочнике вопросов - пункт "Вопросы" в меню наверху)

Как создавать новые вопросы

1. Заходим в справочник вопросов (пункт "Вопросы" в меню наверху)

2. Нажимаем на кнопку "Создать новый вопрос"

3. Заполняем форму:

  • Группа: заполнять не обязательно. Если нужно отнести вопрос к какой-то группе, то эту группу нужно сначала создать (Заходим в справочник вопросов - пункт "Вопросы" в меню наверху - кнопка "Создать новую группу вопросов"). Группы вопросов нужны только для удобства администратора опросов. Респонденты их не видят.
  • Название вопроса: рабочее название вопроса, не для респондентов.
  • Текст вопроса: формулировка вопроса для респондентов
  • Тип вопроса: Single Select - выбора одного варианта ответа из нескольких. Multi Select - выбор одного или нескольких вариантов ответа. Numeric Range - выбор числа из заданного диапазона. Выглядит как бегунок. Number - респондент должен ввести число. Text - свободный ответ.

4. Если тип вопроса это Single Select или Multi Select, то для вопроса необходимо выбрать соответствующий ему список вариантов ответа. Для этого: Заходим в справочник вопросов (пункт "Вопросы" в меню наверху). Находим наш вопрос, проходим по ссылке "Параметры" (попадаем на страницу вопроса). Нажимаем на кнопку "Установить список вариантов ответа". Выбираем нужный список, нажимаем кнопку "Сохранить".

ВНИМАНИЕ: мы можем выбрать только тот список вариантов ответа, который уже существует в базе (в справочнике списков вариантов ответа - пункт "Списки вариантов ответов" в меню наверху)

Как создавать списки вариантов ответа

1. Заходим в справочник списков вариантов ответа (пункт "Списки вариантов ответов" в меню наверху)

2. Нажимаем на кнопку "Создать новый список вариантов ответа"

3. Вводим название списка, сохраняем.

4. После сохранения мы оказываемся на странице нового списка. Теперь нужно ввести сами варианты.

5. Нажимаем на кнопку "Добавить вариант". Появляется желтая форма. Заполняем текст варианта и его численный код. Сохраняем.

Как создать опросный лист (назначить респонденту прохождение опроса)

1. Заходим в справочник опросов (пункт "Опросы" в меню наверху)

2. В списке опросов выбираем нужный. Идем по ссылке "Параметры опроса".

3. На странице опроса нажимаем на кнопку "Добавить опросный лист (назначить респонденту прохождение опроса)". Открывается форма. Выбираем респондента, сохраняем.

ВНИМАНИЕ: мы можем выбрать только того респондента, который уже существует в базе (в справочнике респондентов - пункт "Респонденты" в меню наверху)

Как создать респондента

1. Заходим в справочник респондентов (пункт "Респонденты" в меню наверху)

2. Нажимаем на кнопку "Ввести нового респондента"

3. Заполняем форму, сохраняем.
